# Custom Instructions

## Skills Usage

Always proactively use the available skills when relevant to the task:

- **vercel-react-best-practices**: Use when working with React components, hooks, or React-based features. This includes component design, state management, performance optimization, and following Vercel/Next.js best practices.

- **frontend-design**: Use when implementing UI/UX features, styling components, creating layouts, or making design decisions. Apply this skill for visual design, accessibility, responsive design, and frontend architecture.

These skills should be invoked automatically when the context matches their domain, without requiring explicit user requests.

## Astro MCP Server

Always use the Astro MCP server tools before making any changes related to Astro. This includes:
- Modifying Astro configuration files
- Working with Astro components
- Changing routing or pages
- Updating content collections
- Modifying layouts or integrations

The Astro MCP provides specialized tools and knowledge for working with Astro projects, ensuring changes follow best practices and maintain compatibility.
